# http_status

A simple gem to access HTTP statuses.

## Installation

Add this line to your application's Gemfile:

    gem 'http_status'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install http_status

## Usage

HTTPStatus::Ok # gives 200
